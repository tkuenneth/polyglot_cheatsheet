# while loops

## Dart	

```dart
int count = 0;
do {
    print(count++);
} while (count < 4);
```

## Java	

```java
int count = 0;
do {
    System.out.println(count++);
} while (count < 4);
```

## JavaScript

```javascript
let count = 0;
do {
  console.log(count++);
} while (count < 4);
```

## Kotlin	

```kotlin
var count = 0
do {
  println("${count++}")
} while (count < 4)
```

## Swift

```swift
var count = 0
while (count < 4) {
  print("\(count)")
  count += 1
}
```
