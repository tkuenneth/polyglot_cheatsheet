## Dart	

Dart has a C-like `for`-loop:

```dart
for (var i = 0; i < 4; i++) {
    print(i);
}
```

Also, there is a `foreach`-like alternative:

```dart
const numbers = [12,3,14, 5]; 
for (var n in numbers) { 
    print(n); 
} 
```

This example shows `do`-`while`:

```dart
int count = 0;
do {
    print(count++);
} while (count < 4);
```
